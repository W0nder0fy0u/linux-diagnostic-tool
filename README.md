# Linux Diagnostic Tool (C)

A simple Linux system diagnostic tool written in C.  
This project demonstrates basic system support tasks commonly required in Linux-based environments.

## Features
- Network connectivity check using ping
- Display IP configuration
- Show system information
- Write diagnostic logs to a file
- Menu-driven terminal interface

## Technologies Used
- C Programming
- Linux (Arch Linux)
- GCC
- Make
